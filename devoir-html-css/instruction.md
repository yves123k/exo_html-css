

## Le défi

Votre défi est de construire ce site web et de le rendre aussi proche que possible du design fourni.

## Exigences technique:

- Le site doit être entièrement responsive, s’adaptant parfaitement aux différentes tailles d’écran.
- Le code doit être bien structuré, propre et lisible, afin de faciliter la maintenance et l’évolution du projet.
- Le CSS doit être séparé dans un fichier dédié afin de respecter les bonnes pratiques.

## Resources

 - Icônes : Vous trouverez les icônes dans le dossier icons. Pour les intégrer, utilisez la balise <img>.
 - Images : Vous trouverez les images dans le dossier images. Pour les intégrer, utilisez également la balise <img>.


 ## Notes sur 20

 ## Respect du design (6 points)

 - Fidélité au design fourni : 4 pts
 - Respect des espacements, typographies et couleurs : 2 pts

# Responsivité (4 points)

 - Adaptation correcte sur Téléphone : 2 pts
 - Adaptation correcte sur Ordinateurs : 2 pts

## Qualité du code (6 points)

 - HTML bien structuré (sémantique, bonne hiérarchie) : 3 pts
 - CSS propre, organisé, placé dans un fichier séparé : 2 pts
 - Bonnes pratiques (classes claires, évitement des répétitions, cohérence) : 1 pt

## Question - Response (4 points)

- Capacité à expliquer clairement ses choix techniques : 2 pts
- Pertinence et justesse des réponses lors des questions : 2 pts
